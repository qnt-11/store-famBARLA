Mode Kasir

  • Pencarian: Klik kotak Barcode untuk scan, atau ketik nama dan pilih dari daftar.
  • Keranjang: Klik kolom Qty untuk mengisi jumlah. Tombol ADD Pintar otomatis menanggap kosong = 1.
  • Bayar: Isi uang pembeli, klik WA, Simpan (tanpa nota), atau Cetak.

Mode Stok

  • Supplier: Klik (+) untuk tambah. Klik (⋮) untuk mengubah.
  • Barang: Masukkan 2 Barcode (Scan Satuan & Grosir), Harga, dan Stok.
  • Opsi Barang: Klik (⋮) di samping barang untuk Edit atau Retur.

Peringatan Data!

  • Memori HP: Pantau ikon server. Jika indikator merah (90%), segera hapus riwayat lokal!
  • Wajib Backup: Data tersimpan OFFLINE di browser HP. Jika browser dibersihkan, data HILANG.
  • Tutup Toko: Buka menu (⋮) Stok. Wajib klik Upload Cloud dan Save Excel SETIAP HARI!
